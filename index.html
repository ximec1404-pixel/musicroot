Sitio educativo de programación web

MusicRoot

Explora géneros musicales mediante una página principal y una subpágina. El sitio incluye menú dinámico, slider, mensajes personalizados, modo oscuro, modal informativo y validación de formulario con JavaScript.

Ver subpágina de géneros Probar JavaScript

Propósito del sitio

MusicRoot presenta información básica sobre algunos géneros musicales y demuestra el uso de elementos dinámicos implementados con un archivo JavaScript externo.

La página principal resume el proyecto y la subpágina contiene el detalle de los géneros, cumpliendo con la estructura solicitada de página principal y subpágina.

Funciones implementadas

  • Menú desplegable funcional con clic, teclado y eventos de mouse.
  • Slider automático con controles manuales.
  • Mensaje de bienvenida por hora e idioma.
  • Modal con información dinámica.
  • Formulario validado con mensajes accesibles.
